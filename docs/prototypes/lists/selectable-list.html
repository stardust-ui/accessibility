Selectable list

List component where items are actionable. User can navigate to the list and use arrow keys to navigate through the list items. Enter or space execute an action.

With screen reader, user needs to be able to navigate to and inside of the menu using virtual cursor. Screen reader reads the content of the item as well as the position.
