Menu

Menu Bar with div children

Menu bar with submenu. User can use TAB to navigate to the menu and arrow keys to navigate around the menu. Enter and space open submenu or execute action. Esc closes the submenu.

With screen reader, user needs to be able to navigate to and inside of the menu using virtual cursor. Screen reader reads the aria label of the selected menu item

Compared to the ARIA pattern, it uses div element as a child of the li (instead of button) - the reason is that button has children restrictions.

Known issues:

  • up/down key execute action when not on menu with submenuperformClick
