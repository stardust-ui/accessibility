Menu button

Button component that opens a menu (dropdown) after clicked. User can use arrow keys to select the option and ENTER to execute. ESC closes the menu and returns the focus to the button.

With screen reader, user needs to be able to navigate to and inside of the menu using virtual cursor. Screen reader reads the content of the item as well as the position.
